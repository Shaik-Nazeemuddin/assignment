Front-end developer test instructions

  • You have 90 minutes to complete this test.
  • You should not change the HTML source code.
  • The page should be responsive across devices from phone to desktop.
  • JavaScript should be compatible with ECMAScript 5th edition to support IE 11 and above.
  • Use techniques that generally work in IE 11+ browsers and iPhone 5 or newer (but we won’t test this).
  • The font should be sans-serif and the size should be 16px on larger screens and 12px on smaller screens.

Accordion allowing multiple panels to be opened

Initially, the corresponding accordion-panel for the accordion-button with aria-expanded="true" should be open. When the website visitor activates a button in the accordion, the aria-expanded attributes should be updated and the corresponding accordion-panel should open or close. Zero, one, or many accordion panels may be opened at any time.
